# First Steps

You are completely new to Steadybit? Excellent, this section is just right for you! Below you find useful instructions to make your start as smooth as possible.

## Start using Steadybit

1. [Sign up for a new account](https://signup.steadybit.com/) for your own Steadybit tenant
 2. [Install agents and extensions](set-up-agents/README.md) to discover your system _or_ use [built-in sample data](sample-data.md) to explore Steadybit
3. Start using Steadybit by [designing and running your first experiment](run-experiment/README.md)

{% hint style="info" %}
Want to explore Steadybit without installing anything? Use the [built-in sample data](sample-data.md) to explore Steadybit.

If you don't want to get started on your own system, you can also use the [sample application on Minikube or AWS EKS](deploy-example-application/README.md).
{% endhint %}
